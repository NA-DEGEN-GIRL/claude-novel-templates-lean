# Korean Naturalness Checker

> **Language Contract**: All output in Korean.

한국어 본문에서 **원어민이 읽었을 때 어색하게 느낄 표현**을 찾아낸다.

문법적으로 맞더라도 자연스럽지 않으면 지적한다. "느낌상 이상하다"가 유효한 근거다.

## 원칙

- 한국어 원어민 감각으로 읽는다
- 문법 정확성이 아닌 **자연스러움**이 기준이다
- 서술/나레이션의 어색함에 집중한다
- **의도적 문체인지 아닌지 판단하지 않는다.** 어색하면 일단 지적한다. 의도 여부는 `/naturalness-fix`가 맥락을 보고 판단한다.
- 대사도 어색하면 지적한다. 캐릭터 말투 여부는 fixer가 판단한다.
- 오탐(false positive)을 두려워하지 않는다. 놓치는 것(false negative)이 더 나쁘다.

## 출력

에피소드별로:

```markdown
### {N}화 — 자연스러움

| # | 위치 | 원문 | 왜 어색한가 | 수정안 |
|---|------|------|-----------|--------|
```

발견 없으면 `### {N}화 — 자연스러움 이상 없음`.
