naturalness 보고서(Claude + GPT)의 지적 사항을 **소설의 맥락을 이해한 상태에서** 선별 수정한다.

## 주의

naturalness 체커는 의도적으로 맥락 없이 판단한다. 따라서 보고서에는 **오탐(false positive)**이 포함될 수 있다:
- 캐릭터의 의도된 말버릇이나 문체
- 시적/실험적 표현
- 세계관 고유 용어나 화법
- 의도적 반복/라이트모티프

**이런 것들을 무분별하게 수정하면 소설의 개성이 죽는다.**

## 사용법

`/naturalness-fix` 또는 `/naturalness-fix N-M`

## 사전 조건

1. `summaries/naturalness-report.md`가 있어야 한다. 없으면 `/naturalness` 먼저 실행.
2. `EDITOR_FEEDBACK_gpt_naturalness_chapter-{NN}.md` 파일이 있으면 함께 참조 (GPT 추가 발견). 화별로 개별 파일이 생성되므로, 해당 화의 파일을 찾아 읽는다. 레거시 `EDITOR_FEEDBACK_gpt_naturalness.md` 단일 파일도 호환 지원.
3. `CLAUDE.md`, `settings/01-style-guide.md`, `settings/03-characters.md`를 읽어 문체/캐릭터를 파악한다.

## 처리 방식

> **10화 단위 배치로 처리한다.**

1. `summaries/naturalness-report.md`에서 해당 화수의 지적 사항을 추출한다
   - Claude 검사 항목과 GPT 추가 발견 항목을 **모두** 처리 대상으로 포함
   - 중복 지적(같은 줄, 같은 표현)은 하나로 병합
2. 해당 에피소드를 읽고, 각 지적을 판정한다
3. 수정 적용 + 로그 기록
4. 다음 배치로 넘어간다
5. 배치마다 중간 커밋한다

## 판정 기준

### Step 1. 맥락 확인

해당 에피소드를 읽고, 지적된 표현의 **전후 문맥**을 확인한다.

### Step 2. 판정

| 판정 | 기준 | 조치 |
|------|------|------|
| **수정** | 서술/나레이션에서 정말 어색하고, 의도된 문체가 아님 | 전후 문맥에 맞게 수정 |
| **기각** | 캐릭터 고유 말투, 시적 표현, 의도적 문체, 세계관 용어, 의도적 반복 | 수정하지 않음. 로그에 기각 사유 기록 |
| **보류** | 의도인지 실수인지 판단 어려움 | 보류. 로그에 기록 |

### Step 3. 수정 시 주의

- 보고서의 수정 방향/예시를 그대로 따를 필요는 없다. **문맥에 맞는 더 나은 표현**이 있으면 그걸 쓴다.
- 수정 후 해당 문단을 다시 읽어 흐름이 자연스러운지 확인한다.
- 캐릭터 대사 안의 지적은 해당 캐릭터의 말투 설정을 확인한 후 판단한다.
