# 에피소드 구조

> 모든 에피소드는 이 구조를 기본으로 따른다. 장면 배분은 유동적이되, 전체 구조는 유지한다.

---

## 1. 5막 구조

이 구조는 권장 기본값이다. 장면 흐름이 다른 비율을 요구하면 벗어날 수 있으나, EPISODE_META의 intentional_deviations에 기록한다.

| 구간 | 비율 | 역할 | 핵심 |
|------|------|------|------|
| **Hook** | ~10% | 독자를 잡는다 | 강렬한 첫 문장 또는 상황 |
| **Setup** | ~20% | 상황을 설정한다 | 이번 화의 갈등/목표 제시 |
| **Confrontation** | ~40% | 본편을 전개한다 | 주요 사건, 갈등 심화 |
| **Climax** | ~20% | 정점을 만든다 | 감정/액션의 절정 |
| **Cliffhanger** | ~10% | 다음 화로 이끈다 | 미해결 상태로 마무리 |

### 각 구간 상세

#### Hook (도입)
- 첫 3문장 안에 독자의 관심을 끈다.
- 방법: 긴박한 상황, 의문을 던지는 대사, 감각적 묘사, 이전 화 직후 이어지기
- **지양**: 정지된 설명만으로 길게 시작하지 않는다. 단, 분위기 구축 자체가 목적이면 허용.

#### Setup (전개 준비)
- 이번 에피소드의 핵심 갈등을 명확히 한다.
- 필요한 인물을 등장시키고 현재 상황을 정리한다.
- 이전 화의 여운을 자연스럽게 이어받는다.

#### Confrontation (전개)
- 에피소드의 핵심 내용이 여기에 집중된다.
- 갈등이 심화되고, 장애물이 등장한다.
- 예상 밖의 전개, 정보 갱신, 감정 전환 중 하나를 두는 것을 권장한다.
- 긴장과 이완의 리듬을 만든다.

#### Climax (절정)
- 감정적 또는 물리적 최고조.
- 이번 화의 핵심 질문에 대한 (부분적) 답을 준다.
- 모든 장면이 이 순간을 향해 수렴해야 한다.

#### Cliffhanger (마무리)
- 모든 것을 해결하지 않는다.
- 새로운 의문, 위기, 반전으로 끝낸다.
- "다음 화가 궁금하다"는 감정을 만든다.
- 절벽형 엔딩이 아니어도, 다음 화를 읽을 이유(여운, 미해결 감정, 새로운 의문)를 남긴다.
- **유형**: 위기 직면, 새 인물 등장, 비밀 발각, 선택의 기로

## 2. 장면 전환 규칙

### 시간 경과

```markdown
***

(시간 경과 또는 장면 전환)
```

- `***` (수평선)으로 장면을 구분한다.
- 시간 점프가 크면 첫 문장에서 시간 정보를 자연스럽게 제공한다.
  - ✅ "다음 날 아침, 눈을 떴을 때~"
  - ❌ "[다음 날 아침]" (메타적 표현 금지)

### 장면 수

- 에피소드당 장면: 기본값 2~4개. 장면극(1개)이나 복합 전개(5개 이상)도 허용하되 EPISODE_META의 intentional_deviations에 기록.

## 3. 에피소드 유형별 템플릿

### Type A: 전투/액션

```
Hook  → 위기 상황 직면
Setup → 상대/상황 파악
Confrontation → 긴장 전개 (공세와 후퇴, 상황 역전)
Climax → 결정적 순간 / 결과
Cliffhanger → 새로운 위협 등장 or 대가 지불
```

### Type B: 대화/관계

```
Hook  → 의미심장한 대사 또는 만남
Setup → 대화의 배경/맥락 설정
Confrontation → 갈등 표출, 진심 드러남
Climax → 관계 변화의 순간
Cliffhanger → 관계의 새로운 국면
```

### Type C: 탐색/발견

```
Hook  → 단서 발견 or 의문 제기
Setup → 탐색 시작, 목표 설정
Confrontation → 장애물, 잘못된 단서, 추리
Climax → 핵심 발견 or 진실 도달
Cliffhanger → 발견이 가져올 파장
```

### Type D: 일상/성장

```
Hook  → 평화로운 일상 속 작은 사건
Setup → 캐릭터의 현재 상태/고민
Confrontation → 일상 속 갈등, 자기 대면
Climax → 깨달음 또는 성장의 순간
Cliffhanger → 일상을 깨는 외부 사건
```

## 4. 분량 가이드

| 항목 | 기준 |
|------|------|
| 목표 분량 | {{TARGET_LENGTH}} (예: 4000~6000자) |
| 권장 범위 | 목표의 80~120% |
| 허용 범위 | 목표의 70~130% (사유를 EPISODE_META의 intentional_deviations에 기록) |
| 극단적 이탈 | 70% 미만 또는 130% 초과는 의도와 효과가 분명할 때만 허용 (EPISODE_META에 기록) |
| 대화/묘사/내면 비중 | 장르·시점·에피소드 목적에 맞춰 자연스럽게 배분 |

## 5. 에피소드 번호 규칙

- **프롤로그**: 1번부터 시작, 본편 진입 전까지
- **본편**: 프롤로그 이후 연번 (프롤로그가 5화까지면, 본편은 6화부터)
- **파일명**: `chapter-{{NUMBER}}.md` 형식 (또는 소설별 규칙)
